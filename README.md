# FIFA_World_Cup_Tournament_Manager

fiva world cup tournament manager website using JavaScript,HTML,CSS

## 📌 Repository Description

A football tournament management system inspired by the FIFA World Cup, developed using web technologies to simulate tournament organization, team management, and match scheduling.

This project focuses on creating an interactive system for managing football tournaments with a clean user interface and dynamic game logic.

The application allows users to:

- Generate tournament structures
- Manage teams and matches
- Simulate tournament progression
- Organize match scheduling
- Track tournament flow dynamically
